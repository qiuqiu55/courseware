## 📊 词汇数据说明

### 数据来源
- **文件**：`zhongkao-vocabulary.csv`（86KB）
- **来源**：https://github.com/busiyiworld/maimemo-export
- **内容**：上海市初中英语考纲词汇

### 数据统计
- **总词数**：2208个（比1785词多，可能是版本差异）
- **词性分布**：
  - 名词（n.）：1253个
  - 动词（v.）：65个
  - 形容词（adj.）：207个
  - 副词（adv.）：76个
  - 介词（prep.）：49个
  - 其他：558个

### 数据格式
每个词汇包含：
```javascript
{
  "word": "able",
  "phonetic": "/able/",  // 简化版，实际需词典API
  "pos": "adj.",
  "meaning": "能；有能力的；能干的",
  "collocation": "be able; very able",  // 简化版
  "example": "This is an example...",  // 简化版
  "transformation": {  // 简化版
    "noun": "ation",
    "adverb": "ably"
  }
}
```
