# 中考词汇精讲 Unit 1 教学设计

## 课程信息
- **课程ID**: zhongkao-vocab-unit1
- **适用对象**: 五年级升六年级学生（自学）
- **词汇范围**: 中考考纲前30词
- **学习时长**: 约45分钟（4轮 × 每轮10-15分钟）
- **设计模式**: 四轮学习循环系统

## 设计理念

### 核心问题锚点
**"如何系统化地掌握中考词汇，而不只是死记硬背？"**

### 四轮学习循环设计
基于**记忆心理学**和**深度学习理论**，设计四轮递进式学习：

1. **Round 1 - 词义识别**（认知层）
   - 目标：建立词汇的**音-形-义**连接
   - 方法：AI精讲 + 例题展示 + 互动选择
   - 时长：15分钟

2. **Round 2 - 拼写训练**（记忆层）
   - 目标：强化**字形记忆**和**拼写准确性**
   - 方法：字母拖拽游戏 + 填空练习 + 即时反馈
   - 时长：15分钟

3. **Round 3 - 词形变化**（理解层）
   - 目标：掌握**词汇网络**和**构词规律**
   - 方法：变化规则讲解 + 句型转换 + 易错提示
   - 时长：10分钟

4. **Round 4 - 综合运用**（应用层）
   - 目标：在中考**真实语境**中运用词汇
   - 方法：真题演练 + 错误分析 + 迁移指导
   - 时长：5分钟

## 互动设计细节

### 1. AI精讲模块
- **触发方式**: 点击🤖 AI学伴按钮
- **讲解内容**:
  - 词义详解（中英对照）
  - 常用搭配和固定用法
  - 中考真题例句（带音频）
  - 易混词对比（如 able vs. capable）
  - 记忆技巧（词根词缀、联想记忆）

### 2. 拼写训练游戏
- **游戏机制**:
  - 字母块随机排列，学生拖拽组词
  - 提供"提示"按钮（显示首字母）
  - 错误拼写实时纠正
  - 拼写速度计时（激励挑战）

### 3. 词形变化练习
- **变化类型**:
  - 名词 → 形容词（ability → able）
  - 形容词 → 副词（able → ably）
  - 肯定 → 否定（able → unable）
  - 动词时态变化（如 applicable → applied）

### 4. 错题本系统
- **存储方式**: localStorage（浏览器本地）
- **错题分类**:
  - 词义错误
  - 拼写错误
  - 变化错误
  - 运用错误
- **复习提醒**: 完成单元后自动生成错题复习计划

## 技术实现要点

### 标准模块接入
1. **AI学伴**: 使用TeachAny标准AI学伴API
2. **TTS语音**: 每个词汇提供标准发音（英音+美音可选）
3. **进度追踪**: 使用localStorage存储学习进度

## 评估与反馈

### 形成性评估
- 每轮结束后显示**正确率**
- 错题自动加入**复习队列**
- 提供**学习建议**（如"建议加强拼写练习"）

### 总结性评估
- 单元完成后生成**学习报告**
- 包含：完成时间、正确率、错题分析、下一步建议
